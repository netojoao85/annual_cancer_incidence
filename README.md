# Annual Cancer Incidence      
To gain a better understanding of the incidence of cancer in NHS Borders across Scotland and help inform the planning for the provision of cancer treatment services, was developed an exploratory analysis using Public Health Scotland (PHS) open data. Data cleaning and transformations were carried out to prepare the data for visualisation.


**Links to data:**  
<ul>
 <li> Cancer incidence data - https://www.opendata.nhs.scot/dataset/annual-cancer-incidence </il>
